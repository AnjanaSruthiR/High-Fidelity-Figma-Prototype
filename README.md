# 🪩 VibeCheck - Event Discovery & Ticketing App UI/UX

## 📝 Overview
VibeCheck is a high-fidelity UI/UX design project for a modern event discovery and ticketing platform. Unlike traditional event apps, VibeCheck allows users to discover local activities based on their current "mood" (e.g., Energetic, Chill, Creative) and features social elements like post-event memory galleries.

This project was built to demonstrate proficiency in modern UI design principles, interactive prototyping, and scalable design systems using **Figma**.

## ✨ Core User Flows
This prototype includes fully wired, multi-step user flows featuring start and end points, functional back navigation, and micro-interactions.

1.  **Authentication & Onboarding:** A smooth entry experience featuring a smart-animated onboarding carousel, sign-up/sign-in flows, and dynamic logged-in states.
2.  **Mood-Based Discovery:** Users can horizontally scroll through "Vibe" chips to filter the vertical feed of events based on their current mood.
3.  **Ticketing & Checkout:** A multi-step booking process including ticket quantity selection, dynamic price calculation, and a simulated payment gateway, ending with a digital QR ticket.
4.  **Host a Vibe (Creator Flow):** A portal for users to create, preview, and publish their own local events to the global feed.
5.  **Post-Event Memory Gallery:** Once an event concludes, the digital ticket transforms into a collaborative photo gallery where attendees can upload memories.

## 🛠️ Design System & Technical Execution
This project was built mimicking a component-driven architecture (similar to React) to ensure consistency and ease of handoff to developers.

*   **Components & Variants:** Utilized extensively for Navigation Bars, Input Fields, Buttons, and Mood Filter Chips (handling Default, Hover, and Selected states).
*   **Auto Layout:** Implemented across all screens to ensure responsive, perfectly padded lists, horizontal scrolling chips, and dynamic card heights.
*   **Smart Animate & Transitions:** Used to create fluid, intuitive navigation between screens, overlay modals for checkout, and expanding interactions.
*   **Plugins Utilized:** 
    *   *Unsplash:* For high-quality, authentic event imagery.
    *   *Iconify:* For consistent, scalable vector icons (Lucide/Feather icon sets).
    *   *Storyset:* For engaging onboarding illustrations.
